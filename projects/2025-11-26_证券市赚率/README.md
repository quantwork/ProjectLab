# 市赚率操作手册

>  备注：年末价版

## 一、适用范围与核心公式

**适用标的**

- 高 ROE、分红稳定、业务清晰的 A 股个股
- 典型行业：白酒/消费龙头、优质制造等
- 不适用：强周期（资源、地产）、高杠杆金融、纯故事型成长股

**核心公式（年度市赚率）**

- 年度 PE：
  $$
  PE_Y = \frac{P_Y}{EPS_Y}
  $$

- 年度市赚率：
  $$
  PR_Y = \frac{PE_Y}{ROE_Y^\%}
  $$

其中：

- $EPS_Y$：年报“基本每股收益(元)”（优先扣非）
- $ROE_Y^\%$：年报“净资产收益率(%)”（优先扣非、摊薄），用百分数代入（20% 填 20）
- $P_Y$：**当年最后一个交易日（通常为 12 月 31 日，如遇休市则取最近一个交易日）收盘价**

> 解读：$P_Y$ 视为市场在年末、基于当期信息集对公司全年及中短期基本面的预期定价，
>  以事后披露的 EPS、ROE 进行评估，属于“预期视角 + 事后校验”的年度估值快照。

------

## 二、口径与数据来源（记住这几条）

1. **年报为基准**
   - 只用**年报 ROE(%)**、年报 EPS(元) 做年度 PR
   - ROE、EPS 口径：归母、摊薄，优先“扣非”
2. **PE 取值规则（年末价版）**
   - 不再使用年报公告日 R，而是：
   - $P_Y$ = 当年最后一个交易日（自然年 12 月 31 日，如为非交易日则取之前最近一个交易日）的收盘价
   - $PE_Y = P_Y / EPS_Y$
3. **异常处理**
   - $EPS_Y \le 0$ → 不算 PR（该年退出市赚率体系）
   - $ROE_Y \le 0$ → 不算 PR
   - $PE_Y > 80$ 或 $ROE_Y > 60$ → 标记异常，人工复核再决定是否纳入
4. **数据源示例**
   - 年报：巨潮资讯 / 交易所官网
   - EPS、ROE：东方财富 F10「业绩报表 / 财务指标」或 Wind / Choice 字段
   - 价格：日线收盘价（行情软件 / TuShare / Wind 等），取每年 12 月 31 日或最近交易日收盘价

------

## 三、PR 档位与含义（默认值）

> 按单只高 ROE 个股长期持有设计，可按行业微调

| 档位 | PR 区间        | 含义              | 简要动作倾向                   |
| ---- | -------------- | ----------------- | ------------------------------ |
| A    | PR ≤ 0.4       | 明显低估 / 好价区 | 高仓区：可逐步加到标的上限     |
| B    | 0.4 < PR ≤ 0.7 | 合理略低 / 偏便宜 | 常规仓：正常持有、逢低小幅增持 |
| C    | 0.7 < PR ≤ 1.0 | 合理 / 略偏贵     | 减少增持：逢高可适度减仓       |
| D    | PR > 1.0       | 明显偏贵 / 透支   | 减仓区：仅留底仓或退出         |

------

## 四、年度操作流程（每年标准动作）

对每一只纳入标的，每年执行一次：

1. **整理当年年报数据**
   - 记录：年份 Y
   - 从年报 / 数据源抄：$EPS_Y$、$ROE_Y^\%$
2. **锁定 PE（年末收盘）**
   - 确定自然年最后一个交易日（12 月 31 日或最近交易日）
   - 取 $P_Y$ = 该日收盘价
   - 算：$PE_Y = P_Y / EPS_Y$
3. **计算年度 PR**
   - $PR_Y = PE_Y / ROE_Y^\%$
   - 对照档位（A/B/C/D），记录到「市赚率年表」
   - 年表必含字段：年、EPS_Y、ROE_Y%、P_Y、PE_Y、PR_Y、档位
4. **按档位调整仓位（示例逻辑）**
   - A 档（≤0.4）：
     - 若基本面正常 → 分批加仓至目标高仓（如组合 8–12%）
   - B 档（0.4–0.7）：
     - 保持常规仓位（如 5–8%），可小幅择机增持
   - C 档（0.7–1.0）：
     - 停止新增仓，逢高减到常规下限（如 3–5%）
   - D 档（>1.0）：
     - 分批减仓，保留底仓（如 1–3%），极端情况可清仓

------

## 五、年内信息的使用（简单规则）

- **半年报**：
  - ROE 明显跌破长期中枢（例如 22% → 12%）：
    - 暂停基于 PR 的加仓，先减到安全仓位，等年报再判断。
  - ROE 持平或上升：
    - 维持原估值档位判断。
- **季报（Q1、Q3）**：
  - 只作“预警灯”：盈利 / 现金流严重异常，允许在 PR 仍低的情况下先减部分仓。

------

## 六、注意事项（边界提示）

1. 市赚率只适用“长期高 ROE、分红稳定”的价值股，不适合作周期、故事股。
2. PR 阈值（0.4 / 0.7 / 1.0）为经验参数，可结合利率环境和该股历史 PR 分布动态调整；年末价版 PR 属于“预期视角 + 事后校验”的估值快照，在严格回测或实时信号中可视情况改用公告日口径。
